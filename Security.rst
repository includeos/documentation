.. _Security:

Security
========

If you have found a security issue in IncludeOS please avoid the public issue tracker. Instead send an email to security@includeos.org. If you would like to encrypt your mail please use `this key <https://github.com/includeos/documentation/blob/master/_static/IncludeOS_pgp_public_key.asc>`__ also available on most public `PGP key servers <https://pgp.mit.edu/pks/lookup?search=includeos&op=index>`__.
